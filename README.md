# 🎓 Student Complaint & Feedback Management System

## 📖 Project Description
This project is a responsive front-end web application that allows students to submit complaints and provide feedback digitally.  
Administrators can monitor, filter, and resolve issues through a dashboard.

---

## 🚀 Features
- Student & Admin Login (Role-Based Access)
- Complaint Submission System
- 5-Star Feedback Module
- Admin Dashboard with Complaint Statistics
- Filter Complaints by Status (Pending / Resolved)
- Fully Responsive Modern UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

---

## 🎯 Objectives
- Digitize complaint handling
- Improve transparency
- Reduce paperwork
- Faster issue resolution
